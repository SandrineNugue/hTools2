## vmetrics

The `vmetrics` lib contains basic vertical metrics information for all fonts in the project.

    vmetrics_lib = {
        'default' : {
            'descender' : 300,
            'sc-height' : 700,
            'capheight' : 900,
            'ascender' : 800,
            'units-per-em' : 1000,
            'xheight' : 600
        },
        '15' : {...},
        '55' : {...},
        '95' : {...}
    }
